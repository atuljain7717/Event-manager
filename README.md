# AI Event Manager

## Overview
The AI Event Manager is a web application designed to help users create, manage, and organize events efficiently. This application provides a user-friendly interface for event creation, RSVP management, and event categorization.

## Project Structure
```
ai-event-manager
├── src
│   ├── index.html          # Main entry point of the application
│   ├── landing.html        # Landing page with introduction and links
│   ├── data
│   │   └── events.sample.json  # Sample event data for testing
│   ├── scripts
│   │   ├── app.js          # Main application logic
│   │   └── landing.js      # JavaScript for landing page interactions
│   ├── styles
│   │   ├── main.css        # Main styles for the application
│   │   └── landing.css     # Styles specific to the landing page
│   └── utils
│       └── storage.js      # Utility functions for local storage management
├── .gitignore              # Files and directories to be ignored by Git
├── package.json            # npm configuration file
└── README.md               # Documentation for the project
```

## Features
- Create and manage events with ease.
- RSVP functionality for event attendees.
- Filter and search events based on various criteria.
- Responsive design for optimal viewing on different devices.

## Getting Started
1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd ai-event-manager
   ```
3. Open `src/index.html` in your web browser to start using the application.

## Usage
- Use the landing page to learn more about the application and navigate to the main event manager interface.
- Create new events, manage existing ones, and view statistics on event attendance.
